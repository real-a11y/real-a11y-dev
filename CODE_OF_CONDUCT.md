# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — filing issues, sending pull requests, posting in discussions, or contributing in any other form — you agree to uphold it.

## Reporting

Report unacceptable behavior privately to **conduct@real-a11y.dev**.

Reports are handled confidentially. You can expect an acknowledgement within 72 hours and an outcome (warning, temporary ban, permanent ban, or no action with explanation) within two weeks.

## Scope

This Code of Conduct applies in all project spaces — the GitHub repository (issues, PRs, discussions, commit messages, review comments), the project website, any future chat or mailing-list channels, and when an individual is representing the project in public.
